Política de Privacidad — deFrente Colombia

Última actualización: Marzo 2026

1. Información que recopilamos

deFrente Colombia es una página informativa de contenido político colombiano. No recopilamos datos personales de usuarios directamente.

Utilizamos la plataforma de Facebook/Instagram para publicar contenido. Meta puede recopilar datos según sus propias políticas de privacidad.

2. Uso de la información

El contenido publicado en nuestras redes sociales es de carácter informativo y periodístico sobre política colombiana. No utilizamos los datos de interacción de usuarios para fines comerciales directos.

3. Datos de terceros

Utilizamos APIs de terceros (Facebook Graph API, YouTube Data API) únicamente para publicar contenido. No compartimos datos de usuarios con terceros.

4. Contacto

Para preguntas sobre privacidad: defrentecolombia@gmail.com
